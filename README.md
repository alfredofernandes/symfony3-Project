Symfony3 Project
=============


Docker
-----
- Nginx, Mysql, PHP 7.1, PHP 7.1 Testing.


Installation
-----

#### Docker
```sh
$ docker-compose up
```

#### Local hosts
```
docker_machine_ip symfony3.dev
docker_machine_ip www.symfony3.dev
```
